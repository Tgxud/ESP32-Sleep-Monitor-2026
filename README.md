# ESP32-Sleep-Monitor-2026
基于ESP32与无线传感的非接触睡眠状态监测系统 | 大创项目

项目简介
本项目为大学生创新创业训练计划项目，使用ESP32主控模块 + 超声波传感器，实现非接触式睡眠体动监测、入睡判断与睡眠质量分析。

核心功能
- 非接触式人体微动监测，无需穿戴设备
- 三层信号滤波：过滤过近/过远数据、过滤大幅度突变干扰
- 体动统计与睡眠状态判断
- 稳定输出2cm~300cm范围内的有效距离数据

硬件接线
- Trig 引脚 → GPIO25
- Echo 引脚 → GPIO33
- 供电：5V / GND

项目文件
- `main.py`：主程序，包含信号采集、滤波与体动判断逻辑
- `README.md`：项目说明文档

应用场景
睡眠健康监测、智能养老陪护、大学生创新创业训练计划
